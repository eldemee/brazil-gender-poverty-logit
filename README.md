# brazil-gender-poverty-logit

Group project for Introduction to the Economics of Development course in KU Leuven. My contribution: econometric modeling, regression implementation in R, interaction specification, analysis, plots and interpretation of results.

## Objective of the project
Analyze whether the difference in education between men and women affects the differences in poverty risk.

## Methodology
Binary outcome (poverty status) through a generalized linear model with binomial distribution and logit link.
